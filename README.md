# Excel-Sales-Dashboard# Sales Dashboard

## Project Overview
This project is an interactive Sales Dashboard created in Microsoft Excel to analyze sales performance and identify business trends. The dashboard provides insights into sales, profit, customer behavior, and product performance through dynamic visualizations.

## Key Features
- Total Sales Analysis
- Profit Analysis
- Monthly Sales Trends
- Category-wise Performance
- Region-wise Sales Distribution
- Top Selling Products
- Interactive Filters and Slicers

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Data Visualization

## Insights
- Identified top-performing products and categories.
- Analyzed sales trends across different periods.
- Compared regional sales performance.
- Tracked overall business growth and profitability.
